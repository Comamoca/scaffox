## 🚀 How to use

```
# Run from URL
deno run -A https://raw.githubusercontent.com/Comamoca/scaffox/main/main.ts
```

## ⬇️  Install

```sh

```

```sh
clone https://github.com/comamoca/scaffox
deno run -A ./main.ts
```
